## Examples

#### Hello world example:

```
import ("console");

console.println("hello world!");

# Prints "hello world!"
```

#### Fibonacci example:

```
import ("console");

function fibonacci(n) {
	if (n < 2){
		return 1;
	} else {
		return fibonacci(n-2) + fibonacci(n-1);
	}
}

console.println(fibonacci(7));

# Prints 21
```
